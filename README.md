# Java ATM Simulator

## Overview
A robust, console-based Automated Teller Machine (ATM) simulator built entirely in Core Java. This project demonstrates fundamental Object-Oriented Programming (OOP) concepts, secure user authentication, and real-time session management through a clean command-line interface.

## Key Features
* **Secure Authentication:** Validates user access using predefined Account Numbers and PIN credentials.
* **Transaction Management:** Supports simulated cash deposits and withdrawals with logical error handling (e.g., insufficient funds).
* **Balance Tracking:** Provides real-time account balance updates immediately following any successful transaction.
* **Interactive CLI:** A user-friendly, menu-driven command-line interface for seamless navigation.

## Tech Stack
* **Language:** Java (JDK 8 or higher)
* **Core Concepts:** Object-Oriented Programming (Encapsulation, Abstraction), standard I/O (Scanner), and Control Flow.

## Project Structure
* `ATM.java` — The main entry point that initialises the application.
* `OptionMenu.java` — Handles the user interface, input routing, and menu displays.
* `Account.java` — Manages the underlying bank account logic, secure credentials, and balance calculations.
